## Overview

AutoCAD Professional Suite is a complete design platform with specialized toolsets, collaboration, and automation for professional CAD workflows.

Specialized toolsets for architecture, mechanical, electrical, and MEP provide discipline-specific tools, symbols, and standards that accelerate work in each design field.

Cloud collaboration enables shared drawing access with markup, version comparison, and team review through web and mobile applications.

AutoLISP and.NET scripting automate repetitive drafting tasks, create custom commands, and build workflow integrations.

AutoCAD Professional Suite is the CAD platform for design professionals who need comprehensive drafting capability with the specialization, collaboration, and automation that professional design production demands.

## Performance & Integration

AutoCAD is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while AutoCAD processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows

## System Requirements

- macOS 12 Monterey or later
- Apple Silicon or Intel — Universal Binary
- Works with macOS Ventura, Sonoma and Sequoia
